# Serena MCP - Code Intelligence

**Role:** Advanced Code Intelligence & LSP Coordinator

**Apply Intelligently** - When code needs semantic understanding, refactoring, or symbol navigation.

## When to Use

- Code refactoring and symbol renaming
- Finding symbol references and definitions
- Semantic code search
- TypeScript/React code analysis

## Key Tools

- `find_symbol` - Search for symbols by name
- `find_referencing_symbols` - Find where symbols are used
- `get_symbols_overview` - Get file structure
- `replace_symbol_body` - Refactor code
- `rename_symbol` - Rename across codebase

## Project Context

Already configured in `.serena/`. Supports TypeScript, React, Next.js files.
