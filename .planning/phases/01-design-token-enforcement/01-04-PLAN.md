From lib/design-tokens.ts — token mapping reference:

SPACING TOKENS:
- sectionPadding: { xs: "py-4 md:py-6", sm: "py-6 md:py-8", md: "py-8 md:py-12 lg:py-16", lg: "py-12 md:py-16 lg:py-20", xl: "py-16 md:py-20 lg:py-24" }
- containerPadding: { xs: "px-2 sm:px-4", sm: "px-4 sm:px-6", md: "px-4 sm:px-6 lg:px-8", lg: "px-4 sm:px-6 lg:px-8 xl:px-12" }
- componentPadding: { xs: "p-2", sm: "p-3", md: "p-4 md:p-6", lg: "p-6 md:p-8", xl: "p-8 md:p-10" }
- gap: { xs: "gap-1", sm: "gap-2", md: "gap-3", lg: "gap-4", xl: "gap-6", "2xl": "gap-8" }

COLOR TOKENS:
- colors.background: { base: "bg-background", elevated: "bg-muted", overlay: "bg-background/80 backdrop-blur-md" }
- colors.foreground: { primary: "text-foreground", secondary: "text-muted-foreground", muted: "text-muted-foreground/80" }
- colors.border: { DEFAULT: "border-border", light: "border-border/50", subtle: "border-border/30" }

MIGRATION RULES:
- py-4/py-6 → sectionPadding.xs or .sm
- py-8/py-10/py-12 → sectionPadding.md or .lg
- py-16/py-20 → sectionPadding.xl or .2xl
- px-4/px-6/px-8 → containerPadding.sm or .md
- p-2/p-3/p-4/p-6 → componentPadding.xs/.sm/.md/.lg
- gap-2/gap-3/gap-4/gap-6/gap-8 → gap.sm/.md/.lg/.xl/.2xl
- bg-white → colors.background.base
- bg-gray-50/bg-gray-100 → colors.background.elevated
- text-gray-900/text-black → colors.foreground.primary
- text-gray-500/text-gray-600 → colors.foreground.secondary
- border-gray-200 → colors.border.DEFAULT or .light

EXCEPTIONS (do NOT migrate):
- Animation pixel values, skeleton placeholders with dark: pairs
- Micro-spacing (space-y-1, space-x-2), inline style pixel values
- Classes already correctly paired with dark: variants
</interfaces>
