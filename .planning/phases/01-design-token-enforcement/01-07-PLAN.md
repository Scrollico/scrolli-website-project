COLOR TOKENS:
- colors.background: { base: "bg-background", elevated: "bg-muted", overlay: "bg-background/80 backdrop-blur-md" }
- colors.foreground: { primary: "text-foreground", secondary: "text-muted-foreground" }
- colors.border: { DEFAULT: "border-border", light: "border-border/50", subtle: "border-border/30" }

MIGRATION RULES:
- Home sections are major page sections — use `sectionPadding` for outer vertical padding, `containerPadding` for horizontal gutters, `gap` for grid/flex spacing
- ArticleCard.tsx: Card component — use `componentPadding` for internal padding, `gap` for element spacing within cards
- Newsletter/Banner components: CTA sections — use `sectionPadding` for section wrapping, `componentPadding` for card-like CTAs
- bg-white -> colors.background.base; bg-gray-50 -> colors.background.elevated

EXCEPTIONS: Animation pixel values, skeleton placeholder color pairs, micro-spacing.
